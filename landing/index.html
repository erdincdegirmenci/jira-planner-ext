Chrome Extension

Jira Bulk Planner

A Chrome Extension that bulk updates Jira issues from Excel or CSV files. Manage Assignee, Story Points, and Planned Week Day fields in one go.

1. Installation

To install the extension in Chrome, follow these steps:

  1. Download the latest release from the GitHub Releases page:
    Releases — Jira Bulk Planner
  2. Extract the jira-bulk-planner-vX.X.X.zip file to a folder on your computer.
  3. Go to chrome://extensions in Chrome.
  4. Turn on Developer mode in the top-right corner.
  5. Click Load unpacked and select the extracted folder.
  6. Pin the extension from the puzzle icon (🧩) in the toolbar.

2. Jira Settings

When you first open the extension, configure your Jira connection. It connects to Jira Server / Data Center via Personal Access Token (PAT).

Issue Key Assignee Email Story Points Planned Week Day

PROJ-124 ayse@jira.com.tr 8 W25
